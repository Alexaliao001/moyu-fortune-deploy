# 摸鱼运势App - 设计规范

## 效果图分析

### 1. 老虎机设计（screenshot_1_main.png）
- **布局**: 3列2行（6格），不是传统的3个滚轮
- **外框**: 圆角矩形，金色/黄色渐变边框，有3D立体感
- **内部**: 奶油色/米白色背景
- **图标**: 使用emoji风格图标（猫、睡觉、咖啡、游戏手柄、床等）
- **拉杆**: 右侧有经典的金色拉杆，红色球头
- **标题**: "今天能摸吗？" 大字白色，在老虎机上方

### 2. 结果卡片设计
- **背景**: 金色/橙色渐变，圆角矩形
- **内容**: 
  - 大字"大吉"（白色）
  - 墨镜emoji + 百分比（如"😎 88%"）
- **文案**: 简洁一句话，如"今天老板不在，放心大胆摸！"
- **底部**: "🏆 摸鱼打败了全国 92% 的打工人！"

### 3. 分享页面（screenshot_2_share.png）
- **标题**: "今日摸鱼运势 🐠"
- **卡片**: 奶油色圆角卡片，柔和阴影
- **内容**: 五星评级 + "Excellent Slacking!"
- **文案**: 两行英文摸鱼建议
- **分享按钮**: 圆形图标（微信绿、微博红）
- **头像选择**: 底部弹出面板，圆形头像

### 4. 黄金摸鱼时段页面（screenshot_3_tips.png）
- **标题**: "黄金摸鱼时段" + 闹钟图标
- **时间**: "14:00-15:30" 金色胶囊标签
- **技巧卡片**: 奶油色圆角卡片，左图右文

### 5. 铜钱图标（icon-512x512.png）
- **设计**: 中国古钱币风格
- **装饰**: 龙纹、祥云纹理
- **中心**: 可爱橙色猫咪头像
- **颜色**: 金色/铜色渐变

## 颜色规范

### 主色调
- **橙色渐变**: #FF8C00 → #FF6B35 → #FF4500
- **金色**: #FFD700, #FFA500, #DAA520
- **奶油色**: #FFF8E7, #FFFAF0, #FDF5E6

### 辅助色
- **白色文字**: #FFFFFF
- **深色文字**: #333333, #666666
- **阴影**: rgba(0,0,0,0.1)

## 字体规范

### 中文
- **标题**: 思源黑体 Bold / 苹方 Semibold
- **正文**: 思源黑体 Regular / 苹方 Regular

### 英文
- **标题**: SF Pro Display Bold
- **正文**: SF Pro Text Regular

## 圆角规范
- **大卡片**: 24px - 32px
- **小卡片**: 16px - 20px
- **按钮**: 12px - 16px
- **图标容器**: 50% (圆形)

## 阴影规范
- **卡片阴影**: 0 4px 20px rgba(0,0,0,0.1)
- **按钮阴影**: 0 2px 8px rgba(0,0,0,0.15)
- **悬浮阴影**: 0 8px 30px rgba(0,0,0,0.2)

## 动画规范
- **老虎机滚动**: 垂直滚动，依次停止
- **铜钱旋转**: 3D翻转效果
- **拉杆**: 下拉回弹效果
- **结果出现**: 淡入+缩放
