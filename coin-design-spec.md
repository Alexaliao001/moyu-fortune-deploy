# 中国古铜钱设计特征

## 核心特征
1. **外圆内方** - 圆形外轮廓，中间有正方形方孔
2. **宽缘** - 外圈有明显的宽边缘（轮廓），凸起的边
3. **方孔** - 中心是正方形的孔，不是圆孔
4. **汉字** - 方孔四周有汉字（如"开元通宝"、"康熙通宝"等），上下左右各一字
5. **铜色** - 古铜色/青铜色，带有岁月感的金属质感

## 颜色
- 新铜钱：金黄色/橙黄色 (#C8963C ~ #D4A840)
- 古铜钱：深铜色/青铜色 (#8B6914 ~ #A07828)
- 高光：淡金色 (#F0D878)
- 暗部：深棕色 (#604810)

## 结构（从外到内）
1. 外缘（凸起的圆形边框）
2. 钱面（平面区域，有汉字）
3. 内缘（方孔周围的凸起边框）
4. 方孔（正方形穿孔）

## 我们的设计方案
- 保持外圆内方的经典造型
- 方孔位置放用户选择的头像（替代传统方孔）
- 外缘用金色渐变模拟凸起效果
- 钱面区域用深铜色+纹理
- 四周可以放简化的装饰纹路（代替汉字）
