**核心功能：**
- 老虎机式抽签，5种运势等级

### 帖子 2：技术分享帖

**标题**：独立开发一个摸鱼运势App的技术栈分享

**正文**：
最近做了一个小产品「摸了么」，分享一下技术栈：

**前端：**
- React 19 + TypeScript
- Tailwind CSS 4
- GSAP 动画（老虎机效果）
- i18next（国际化）
- PWA（Service Worker + IndexedDB）

**后端：**
- Express + tRPC
- Drizzle ORM + MySQL
- LLM API（生成金句）

**特色功能：**
- Canvas 生成分享海报
- 离线缓存支持
- 站内通知系统

**一些思考：**
1. 大吉概率从30%降到5%，增加稀缺性
2. 金句库加入大量反内卷语录，贴合时代情绪
3. 梗图风格海报，更适合社交传播

项目地址：chillworks.ai
